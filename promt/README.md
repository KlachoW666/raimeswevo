# Mini App (Zyphex) — единственное приложение

Здесь всё: фронт, бэкенд и обмен USDT → ZYPHEX.

- **frontend/** — интерфейс (React), в том числе экран «Обмен USDT → ZYPHEX».
- **backend/** — API: авторизация, кошелёк, курс и обмен ZYPHEX (`/api/zyphex/rate`, `/api/zyphex/exchange`).

Папка `Zyphex/` в корне репозитория не используется при деплое и не нужна для работы этого приложения.
